# Archived Conversations

> High-value Slack threads preserved for future reference. Use sparingly — only when the discussion itself has lasting value beyond a specific deal.

## Naming Convention

`YYYY-MM-DD-topic.md`

## When to Archive a Conversation

- The thread produced a strategic insight or decision not captured elsewhere
- A debate or analysis worth revisiting later
- Market intel or relationship context that doesn't fit in a deal folder

_(conversations will be archived here when flagged in Slack)_
