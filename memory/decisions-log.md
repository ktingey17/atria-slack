# Decisions Log

> Running log of key decisions made across all deals and strategy. Newest entries at the top. This is the long-term memory — when someone asks "why did we do X?" or "didn't we already decide Y?", this is where Claude looks.

## How to Use

Each entry should capture:
- **Date** — when the decision was made
- **Deal/Topic** — what it relates to
- **Decision** — what was decided
- **Rationale** — why (even a one-liner)
- **Who** — who made or drove the call
